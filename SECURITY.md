# Security Policy

## Supported Versions

This project is pre-1.0 and developed on the `main` branch. Security fixes are
applied on the latest codebase.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Report vulnerabilities privately to: maintainer@example.com

Please include:

- A clear description of the issue
- Reproduction steps or proof-of-concept
- Potential impact
- Any suggested remediation

You can expect an initial response within 5 business days. After confirmation,
we will work on a fix and coordinate responsible disclosure.
